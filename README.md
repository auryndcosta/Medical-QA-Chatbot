# Medical-QA-Chatbot

# To execute the code in your system:
### STEPS:

Clone the repository

```bash
git clone https://github.com/auryndcosta/Medical-QA-Chatbot.git
```

### STEP 1 - Create a conda environment after opening repository

```bash 
conda create -n medibot python=3.13.9 -y
```

```bash
conda activate medibot
```

### STEP 2 - Install the requirements

```bash
pip install -r requirements.txt
```
